# PerformanceReviewsAnalyser: A Comprehensive Sentiment Analysis and HR Recommendations Tool for Employee Performance Reviews

**Overview**

PerformanceReviewsAnalyser is a Python tool that empowers HR professionals with cutting-edge sentiment analysis and GPT-based modeling capabilities to gain deep insights into employee performance reviews. By automating the categorization of feedback and providing actionable HR recommendations, this tool streamlines the review process and enhances employee development.

**Features**

* **Sentiment Analysis:** Leverages the Vader Sentiment Analyzer to accurately classify feedback into positive, negative, or neutral sentiments, providing a comprehensive understanding of employee perceptions.

* **GPT Model for HR Recommendations:** Incorporates a custom-developed and fine-tuned GPT model to generate intelligent HR recommendations based on the sentiment analysis, identifying areas for improvement and suggesting targeted development strategies.

* **Data Generation:** Simulates employee names and reviews to create a robust dataset for evaluation and testing, ensuring the tool's effectiveness in real-world scenarios.

* **Dashboard for Insights:** Enables the creation of interactive visual dashboards to visually represent analytical results, providing a clear and concise overview of employee performance trends and areas of focus.

* **Data Import/Export:** Supports importing and exporting data in various formats, facilitating seamless integration with existing HR systems and ensuring data portability.

**Installation**

1. **Clone the Repository:** Clone the PerformanceReviewsAnalyser repository to your local machine.

2. **Install Dependencies:** Install the necessary Python libraries using the command `pip install -r requirements.txt`.

3. **Set Up Environment:** Configure the tool's environment settings, including data paths and output preferences.

**Usage**

The PerformanceReviewsAnalyser tool is designed for easy and intuitive use:

1. **Data Generation:** Utilize the data generation functionality to simulate employee reviews and populate the dataset.

2. **Sentiment Analysis:** Run sentiment analysis using the Vader Sentiment Analyzer to categorize feedback into positive, negative, or neutral sentiments.

3. **HR Recommendations:** Apply the trained GPT model to generate intelligent HR recommendations based on the sentiment analysis.

4. **Visualization:** Create interactive visual dashboards to visualize analytical results and identify key trends.

5. **Data Export:** Export the processed data and generated insights in various formats for further analysis or reporting.
